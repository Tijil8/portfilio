Ten times faster, from memory layout alone

A hand-rolled Entity Component System benchmarked against the naive object-oriented approach doing identical, checksum-verified work. At one million entities the data-oriented version is 10.4× faster — purely from how the bytes sit in memory, not from any change to the algorithm. Comes with a live 3D demo you run in the browser: press a key and watch the memory layout swap under a running simulation.
