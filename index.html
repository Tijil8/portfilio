Same work, 10× faster — a data-oriented ECS, measured

A hand-rolled Entity Component System benchmarked against the naive OOP approach doing identical, checksum-verified work. At one million entities the data-oriented version is 10.4× faster — purely from how the bytes sit in memory, not from any change to the algorithm. Comes with a live 3D demo compiled to WebAssembly: press a key and watch the memory layout swap under a running simulation.
